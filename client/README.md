# 🍽️ Resto - Restaurant Ordering System

A full-stack restaurant ordering web app built with React JS and Express JS.

## 📌 Features
- Browse restaurant dishes with photos and prices
- Add dishes to cart and place orders
- Kitchen dashboard for chef to view live incoming orders
- Real-time order updates using Socket.IO

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| Frontend | React JS |
| Backend | Express JS (Node.js) |
| Database | MongoDB (Mongoose) |
| Real-time | Socket.IO |
| File Upload | Multer |
